this project, I dedicated a significant amount of time to learning how to use virtual machines (VMs), which are software-based environments that simulate a physical computer system. They allow users to run different operating systems on a single device without needing additional hardware. Through this hands-on experience, I successfully created and configured six virtual machines: Windows 11, Ubuntu, Linux Mint, Fossapup, Dragonfly, and macOS.

Each setup gave me insight into how different systems operate, and I gained practical skills in managing virtual environments. Creating a macOS VM was particularly challenging, but after extensive research and troubleshooting, I was able to get it running smoothly. This project deepened my understanding of system architecture, virtualization software, and cross-platform compatibility.
