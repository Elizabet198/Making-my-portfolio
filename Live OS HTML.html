As part of my exploration into different operating systems, I experimented with several live OS environments, including Ubuntu, Windows PE, and Linux. This hands-on experience taught me how to download, install, and run these systems from a bootable drive without affecting the host machine. I also learned to use Rufus, a tool that allowed me to create bootable USB drives to run these live OS environments. Through this process, I gained valuable knowledge about system management, boot configurations, and how live OS systems can be utilized for troubleshooting, recovery, and other specialized tasks. This project significantly expanded my understanding of operating systems and their practical applications.
